What is a Code Retreat?

A code retreat is a day-long event where programmers gather to practice the fundamentals of programming outside of the constraints of work.

Pairs of programmers work to solve a problem in 45 minutes. After 45 minutes, the code is deleted. The programmers perform a retrospective, reflecting on their progress and how things went.

This process is repeated throughout the day, with each iteration having different partners and constraints.

The Global Day of Code Retreat is a worldwide event with code retreats in many major cities.

Why should I go?

There are many benefits to attending a code retreat:

  • Practicing the fundamentals improves your mastery
  • Programming outside the pressures of work gives you the freedom to experiment
  • Working with new programmers teaches you new ways of thinking, new workflows, new programming languages and tools.

What will I need to bring?

  1. A laptop that is ready to write code in your favorite language(s).
  2. A unit testing library, ready to use.
  3. Some sort of offline version control system (Git), if you are familiar with one.
  4. A stopwatch/timer app, if you have a smartphone.
  5. A pen & paper, if it helps you brainstorm.
